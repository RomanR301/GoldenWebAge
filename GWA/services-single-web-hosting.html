WEB HOSTING

Through advice on secure, high-performance and inexpensive hosting

We don't run our own servers. Instead, we will advise you on which hosting solution you need, taking into account the specific requirements of your application. A healthy compromise between price and performance is particularly important to us. We also value modern infrastructure and technologies that are always up-to-date. For example, with Wordpress, Laravel and other PHP applications, the latest version of PHP, MySQL and Apache / Nginx are very important for high security and performance.

We support hosting for all sizes

Of course, we not only advise you on the best solution, but also implement it directly in your application. From simple shared hosting to dedicated, horizontally scaling applications, we have already deployed everything. We use Uberspace or 1und1 hosting for small applications. For large solutions, mostly AWS (Amazon Web Service) or, more rarely, Microsoft Azure or Google Cloud is used. For larger applications that can automatically scale horizontally, we rely on Docker to provide the same services with significantly fewer resources.

What we also offer:

SSL: Encryption of the transmission with SSL certificates for the "s" in https
Htaccess: Redirecting old URI structures, headers for browser-side caching and much more.
Caching: Varnish for HTML caching or Memcached / Redis for caching in the application
